# Recipe Extractor

The Recipe Extractor project is designed to extract structured recipe information such as
ingredients and cooking instructions from input data.
This project is implemented using Python and demonstrated using a Jupyter Notebook.

---

## Project Objective

- Extract recipe details from given input
- Convert unstructured recipe content into structured format
- Demonstrate text processing using Python
- Provide a simple academic-level implementation

---

## Technologies Used

- Python
- Jupyter Notebook
- Natural Language Processing
- Text or Image Processing
- Git and GitHub

---

## Project Structure

recipe-extractor/
|
|-- newonefinalcode.ipynb   Main notebook containing recipe extraction logic  
|-- README.md              Project documentation  
|-- requirements.txt       Required Python libraries  
|-- results/               Output screenshots and results  

---

## How to Run the Project

Step 1: Clone the repository

git clone https://github.com/VETHAPRIYA/recipe-extractor.git  
cd recipe-extractor

Step 2: Install required libraries

pip install -r requirements.txt

or manually install

pip install numpy pandas nltk pytesseract opencv-python

Step 3: Run the notebook

jupyter notebook

Open the file

newonefinalcode.ipynb

Run the cells step by step to see the recipe extraction output.
